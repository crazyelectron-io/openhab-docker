# openhab-docker

Docker files to generate openHAB and supporting services containers

> Secrets are encrypted with git-crypt; decrypt with `git-crypt unlock` after cloning (requires GPG key).

This repository contains Docker files to run my openHAB production system in containers.

- InfluxDB:1.7.9
- Grafana:6.5.1
- Mosquitto:1.6.7
- NefitEasy-server:1.1.0
- openHAB:2.5.0
